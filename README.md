# HR Cheat Sheet PWA

A professional Progressive Web App (PWA) designed to help job seekers prepare for interviews by managing common questions and answers in multiple languages.

## ‚ú® Features

- **Multilingual Support**: Fully localized interface (English and Ukrainian).
- **Dynamic Content**: Add, edit, and delete interview questions.
- **Advanced Job Management**: Create, rename, and delete custom job categories to organize questions by specific roles.
- **Answer Tips**: Store helpful hints for each question to remember key points during preparation.
- **Drag & Drop**: Reorder questions within categories easily to prioritize your preparation.
- **Smart Data Handling**: Automatic migration of questions to a general category when a job title is deleted.
- **Data Portability**: Export your entire database to a JSON file and import it on any device.
- **PWA Ready**: Installable on mobile and desktop devices with offline support.
- **Modern UI**: Accessible interface with Dark/Light mode support and focus management.

## üöÄ How to Use

1. **Manage Jobs**: Click the "Tags" icon to create or edit job categories.
2. **Add Question**: Click the "Add Question" button and link it to a specific job category.
3. **Organize**: Use the drag handle (vertical grip icon) to reorder questions.
4. **Switch Language**: Use the selector in the header to toggle between English and Ukrainian.
5. **Theme**: Toggle between Dark and Light mode using the moon/sun icon.
6. **Backup**: Use the "Export" button to save your data.
